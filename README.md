## 요구사항
#### 기본 구분자: , 나 : 로 구분
- 구분자를 통해 합을 구한다.
#### 커스텀 구분자: //나 \n으로 사이의 값으로 구분
- \n 다음의 값들을 구분자로 분리한다
- 그 합을 구한다.
## 🛠️ 기능구현 목록
### 구분자
- [ ] 기본 구분자는 ,와 : 로 분리한다.
- [ ] 커스텀 구분자는 //와 \n 사이의 값으로 구분한다.
- [ ] 기본구분자인지 커스텀 구분자인지 구별한다
- 입력 값의 맨 앞 두글자가 //이라면 커스텀 구분자이다.
- 그게 아니라면 기본 구분자이다.
